# Explorando IA Generativa em um Pipeline

Este projeto demonstra a integração de inteligência artificial generativa em um pipeline de dados utilizando Python. O objetivo é extrair, transformar e analisar dados, aplicando recursos de IA para enriquecer o processo.

## Funcionalidades
- Leitura e manipulação de dados com pandas
- Utilização da API OpenAI para geração de insights ou automação de tarefas
- Pipeline ETL simples e didático

## Estrutura
- `etl_sdw.py`: Script principal do pipeline ETL
- `santander.csv` e `SDW2025.csv`: Bases de dados utilizadas

## Requisitos
- Python 3.13+
- pandas
- openai

## Como executar
1. Instale as dependências no ambiente virtual:
   ```bash
   pip install pandas openai
   ```
2. Execute o script principal:
   ```bash
   python etl_sdw.py
   ```
